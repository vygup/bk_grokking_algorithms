>`Бинарный поиск` - алгоритм, на вход которого поступает **отсортированный** список, при нахождении требуемого элемента возвращается его позиция, иначе null.

Принцип поиска: каждый раз происходит переход в середину диапазона, тем самым отсекая половину оставшихся чисел с определенной стороны, и так пока не найдется требуемый элемент.

	При бинарном поиске числа из списка [1...100], элемент определяется за 7 шагов. При поиске в [1...240e3], элемент находится за 18 шагов

>Для списка из *n* элементов, бинарный поиск выполняется за *log<sub>2</sub>n*, за *логарифмическое время*

>Бинарный поиск работает только если список отсортирован

Бинарный поиск в `python`
```python
def binary_search(list, item):
# Границы той части списка, в которой выполняется поиск
low = 0
high = len(list)—1

while low <= high:  # Пока граница не сократится до 1 элемента
	mid = (low + high)/2  # Проверяем средний элемент
	guess = list[mid]
	if guess == item:  # Значение найдено
		return mid
	if guess > item:  # Много
		high = mid - 1
	else:  # Мало
		low = mid + 1
	return None
```
